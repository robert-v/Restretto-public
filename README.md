# Restretto - minimal REST client for macOS

## Features
- Native & Free
- Folders
- Import cUrl from clipboard
- History view & Inspector

## Installation

1. Download dmg archive from https://restretto.app
2. Mount the dmg and move .app file to /Applications
3. Run the app
